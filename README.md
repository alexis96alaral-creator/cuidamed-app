# CuidaMed 🏥

**Enfermería profesional a domicilio en Santa Cruz, Bolivia**

Plataforma que conecta familias con enfermeros certificados y verificados.

## 🚀 Cómo correr localmente

```bash
npm install
npm run dev
```

Abrí http://localhost:5173

## 🏗️ Build para producción

```bash
npm run build
```

## 🌐 Deploy

Este proyecto está deployado en Vercel en [alaralcuidamed.com](https://alaralcuidamed.com)

## 📁 Estructura

```
src/
  App.jsx        # Aplicación principal (Home, Buscar, Quiénes somos)
  Perfiles.jsx   # Módulo de perfiles de enfermeros
  main.jsx       # Entry point
  index.css      # Estilos globales
public/
  favicon.svg    # Ícono de CuidaMed
```
